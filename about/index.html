About

Hi all!

I created this blog mostly for sharing some interesting stuff from topics of my interests, like:

  • reversing (mostly Windows binaries),
  • development,
  • my repo script demos - how to use them.

If you find my blog posts useful please let me know by contacting me via comments or by social media listed on contact page.
